  • How many cells do we need to sample so that we see at least n cells of each type? This depends on the number of cell type present and the diversity, i.e. the entropy. When entropy is very low, few or one cell type dominates the population and all other types are present at low fractions. In the default example below we assume that there are 11 cell types. One type is present at 80% and 10 are present at 2% each. If we want to be 95% confident that our sample contains at least 5 cells from each cell type, we need to sample at least 619 cells.

  • The underlying assumption is that for every type the number of cells in the population is much larger than the minimum number of cells desired in the sample. Then, for a given type the probability of seeing at least n cells in a sample of size k follows the cumulative distribution function of a negative binomial NBcdf(k; n, p). For m cell types with the same parameter p the overall probability of seeing each type at least n times is NBcdf(k; n, p)^m. The results below should be treated as a lower bound, since in reality the number of cells in the population is finite and the random draws are not independent per cell type.
